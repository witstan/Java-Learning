# step3:
> 提交修改描述信息
* 1.查看当前修改的状态 - 可选
```
git status
```
* 2.添加修改文件至暂存区
```
git add --all
git add .
```
* 3.添加提交说明信息
``` bash
git commit -m 'xxx提交说明内容'
```

# step 4:
> 推送添加的暂存文件至远端仓库
``` bash
git push
```
